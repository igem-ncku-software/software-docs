# Linux 基礎筆記（WSL 環境）

> 這份筆記整理給剛開始使用 WSL 的新手，涵蓋終端機操作中最基礎、最常用的部分。

## 1. Linux 是什麼？

Linux 是一套作業系統的核心（Kernel），負責管理硬體資源；平常我們透過 **Shell**（例如 Bash）輸入文字指令，跟這個核心溝通、操作電腦。WSL（Windows Subsystem for Linux）讓你可以在 Windows 裡直接執行一份完整的 Linux 環境（預設是 Ubuntu），不用額外安裝虛擬機。

## 2. 終端機基本操作

打開 WSL 後，會看到類似這樣的提示字元（prompt）：

```bash
username@hostname:~$
```

- `username`：目前登入的使用者名稱
- `hostname`：主機名稱
- `~`：目前所在的目錄（`~` 代表家目錄）
- `$`：一般使用者提示符號（如果是 `#` 代表以 root 身份操作，要特別小心）

!!! tip "提示"
    在 `$` 後面輸入指令，按下 `Enter` 就會執行。

## 3. 目錄與路徑觀念

- **絕對路徑**：從根目錄 `/` 開始描述的完整路徑，例如 `/home/username/notes`
- **相對路徑**：從目前所在位置開始描述，例如 `notes/today.md`
- 特殊符號：
    - `~`：代表使用者的家目錄（例如 `/home/username`）
    - `.`：代表目前所在的目錄
    - `..`：代表上一層目錄

## 4. 常用指令

### 4.1 移動與查看

| 指令 | 說明 |
|------|------|
| `pwd` | 顯示目前所在的目錄路徑 |
| `ls` | 列出目前目錄下的檔案與資料夾 |
| `ls -la` | 列出所有檔案（含隱藏檔）與詳細資訊 |
| `cd 目錄名稱` | 切換到指定目錄 |
| `cd ..` | 回到上一層目錄 |
| `cd ~` | 回到家目錄 |
| `mkdir 資料夾名稱` | 建立新資料夾 |
| `touch 檔案名稱` | 建立新的空白檔案 |

### 4.2 複製、搬移、刪除

| 指令 | 說明 |
|------|------|
| `cp 來源 目的地` | 複製檔案 |
| `cp -r 來源資料夾 目的地` | 複製整個資料夾 |
| `mv 來源 目的地` | 搬移或重新命名檔案 |
| `rm 檔案名稱` | 刪除檔案 |
| `rm -r 資料夾名稱` | 刪除整個資料夾 |

!!! warning "小心使用 rm"
    `rm` 刪除的檔案不會進資源回收桶，是直接永久刪除，操作前請務必確認路徑正確。

### 4.3 查看與編輯檔案內容

| 指令 | 說明 |
|------|------|
| `cat 檔案名稱` | 直接在畫面上顯示整個檔案內容 |
| `less 檔案名稱` | 分頁瀏覽檔案內容（按 `q` 離開） |
| `nano 檔案名稱` | 開啟簡單的文字編輯器來編輯檔案 |

!!! tip "nano 常用快捷鍵"
    - `Ctrl + O`：儲存檔案
    - `Ctrl + X`：離開 nano

## 5. 檔案權限基礎

Linux 每個檔案都有三種身份的權限設定：**擁有者（owner）**、**群組（group）**、**其他人（others）**，每種身份各有三種權限：

- `r`（read）：可以讀取
- `w`（write）：可以寫入/修改
- `x`（execute）：可以執行

用 `ls -l` 可以看到類似 `-rwxr-xr--` 的權限字串，分別對應：擁有者 `rwx`、群組 `r-x`、其他人 `r--`。

修改權限可以用 `chmod`：

```bash
chmod +x script.sh   # 幫檔案加上「可執行」權限
chmod 755 script.sh  # 用數字方式設定權限（擁有者7、群組5、其他人5）
```

## 6. 安裝軟體基礎

WSL 預設是 Ubuntu，使用 `apt` 這個套件管理工具來安裝軟體：

```bash
sudo apt update            # 先更新套件清單
sudo apt install 套件名稱   # 安裝指定的軟體
```

!!! note "為什麼要先 update？"
    `apt update` 只是更新「套件清單」（知道有哪些新版本可以裝），不會真的安裝或升級軟體。養成先 update 再 install 的習慣，可以避免裝到過舊的版本資訊。

## 7. 其他實用小技巧

- **`sudo`**：以系統管理員權限執行指令，執行有風險的操作（如安裝軟體、修改系統檔案）時常需要
- **Tab 自動補全**：輸入指令或檔名時按 `Tab` 鍵，可以自動補全，加快輸入速度並減少打錯字
- **`↑` / `↓`**：翻閱之前輸入過的指令歷史紀錄
- **`Ctrl + C`**：強制中斷目前正在執行的指令
- **`clear`**：清空終端機畫面
